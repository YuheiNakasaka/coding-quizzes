# 出題意図

この問題は以下の能力を測定することを目的としています：

1. **Command Patternの理解と実装**
   - 操作のカプセル化
   - 実行と取り消しの分離
   - コマンドオブジェクトの設計

2. **Undo機能の実装スキル**
   - 操作履歴の管理
   - 逆操作の実装方法
   - 状態の保存と復元

3. **文字列操作の実装力**
   - 効率的な文字列の挿入・削除・置換
   - インデックス計算の正確性
   - エッジケースの考慮

4. **実践的なアプリケーション設計**
   - テキストエディタの基本機能
   - 拡張可能なコマンド構造
   - ユーザビリティの考慮

Command Patternは、操作の記録、取り消し、やり直し、マクロ機能など、
多くのアプリケーションで使われる重要なパターンであり、
この実装を通じて実践的な設計スキルを測定します。
